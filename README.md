# 💣 Minesweeper Web

## 📌 Deskripsi

Game Minesweeper sederhana berbasis web yang terinspirasi dari game klasik.
Pemain harus membuka semua kotak aman tanpa mengenai ranjau.

---

## 🎯 Tujuan

* Membuat game yang bisa dimainkan langsung di browser
* Menjadi project portfolio
* Melatih logika dan JavaScript

---

## 👤 Target User

* Pemula belajar coding
* Pengguna yang suka game puzzle ringan

---

## 🎮 Fitur Utama

* Grid 8x8
* 10 ranjau (mine) acak
* Klik kiri untuk membuka kotak
* Klik kanan untuk memberi tanda (flag 🚩)
* Angka menunjukkan jumlah ranjau di sekitar
* Sistem menang & kalah
* Tombol restart

---

## 🔄 Cara Bermain

1. Klik kotak untuk membuka
2. Hindari ranjau
3. Gunakan flag untuk menandai ranjau
4. Buka semua kotak aman untuk menang
